shell自带工具调试

shell自带的调试，用参数 -x 和 -n。

1、 bash -x shell文件
会对整个的shell文件执行。
有+的是代码，没有的echo输出。
在shell脚本中某段代码前后添加set -x和set +x可以只调试此段代码。

2、 bash -n shell文件
只检查脚本是否正确，不真的执行。

bashdb

这是一个第三方工具，安装：sudo apt-get install bashdb
bashdb可以单步的执行，有很多的参数用于调试：

bashdb –debug file 即可开始调试：

其他参考：http://blog.techbeta.me/2015/10/shell-debug/
